# Selectable


## Make <select> more beauty by jQuery


### examples
```javascript
$('select').makeSelect({width:100,className:'small',withSearch:true});
```


### Selects
```html
<select data-width="100">
	<option value="1">Value</option>
</select>
```

```html
<select data-width="100" disabled>
	<option value="1">Value</option>
</select>
```
